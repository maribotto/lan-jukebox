# lan-jukebox
![Screenshot](https://i.imgur.com/9ESDCbO.png)
```
git clone https://github.com/maribotto/lan-jukebox && cd lan-jukebox
```
```
npm install
node server.js
``````
Edit `config.json` and enter your LAN IP address. Open your web browser and enter `localhost:3000`. Others should use your LAN IP address.
